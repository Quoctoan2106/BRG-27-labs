# 1a-1 Virtualisation & Linux Setup

## Lab Objective
Set up a Linux environment on Windows PC using VirtualBox and Ubuntu.

## Setup Instructions
1. Install VirtualBox
2. Download Ubuntu ISO
3. Configure VM (4GB RAM, 20GB disk)
4. Install Ubuntu
5. Configure NAT network
6. Reboot and confirm setup
